Initial release.

- `/confirm` slash command for Claude Code that writes long-form post-purchase confirmation / welcome letters for financial direct-response promotions
- ~1,500–2,500 word output, signed by the editor in first person
- Delivers a formatted Word doc (.docx) on the user's Desktop — Calibri body, bold subheads, italicized publication names, bonus-report download links bolded between arrow brackets
- 5-job structure: welcome → resell the Big Idea → editor credibility → bonus-report links (delivered twice) → "Take Your First Steps Today"
- 3-tier credibility sourcing: shared AIVault Guru Wikis (Dropbox) → submitted promo → web search
- Auto-detects the AIVault Dropbox folder; falls through silently if the user doesn't have access
